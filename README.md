# ⚔️ Capital Wars — 資本戦争ボードゲーム

会社・株・ATMを駆使して最大の資産を築け！  
子どもから大人まで楽しみながら「お金の仕組み」が学べる、1〜10人対応のデジタルボードゲームです。

---

## 🎮 ゲーム概要

| 項目 | 内容 |
|------|------|
| **プレイ人数** | 1〜10人（AI プレイヤー対応） |
| **プレイ時間** | 約 20〜40 分（年数設定で調整可） |
| **対象年齢** | 小学生〜大人 |
| **勝利条件** | ゲーム終了時に **ATM残高 ＋ 保有株価額 ＋ 手持ち現金** が最大のプレイヤーの勝ち |

---

## ✅ 実装済み機能

### ゲームコア
- **1ターン1サイコロ制** — サイコロ1回で全保有会社・株の損益を一括計算
- **サイコロ先振り強制** — 会社/株を持っている場合、サイコロを振るまで他のアクション（購入・ATM・はたらく）がロックされる
- **1アクション制** — サイコロ/ATM/はたらく のいずれか1つを選んだらターン終了

### 会社システム
| 会社 | コスト | 効果 |
|------|--------|------|
| 🍜 飲食店 | 500円 | 1-2: -500 / 3-4: +500 / 5-6: +1500（★3にアップグレード可） |
| ⭐ 三ツ星レストラン | 500円 | 1: -500 / 2-6: +2000 |
| 🚌 バス会社 | 1000円 | 5-6: 追加行動ボーナス（🚄鉄道にアップグレード可） |
| 🚄 鉄道会社 | 1000円 | 5-6: 再ターン |
| 📺 メディア | 2500円 | 1-2: 他プレイヤーから2500円 / 3-4: 1250円 |
| 🏦 金融機関 | 2500円 | 他プレイヤーへ融資・最終ターンに全額回収 |

### 株システム
| 株 | 購入価格 | 偶数 | 奇数 |
|----|----------|------|------|
| 🇯🇵 日本株 | 500円/株 | +1500円 | -1000円 |
| 🌍 外国株 | 1000円/株 | +2500円 | -1500円 |

### 在庫管理（プレイヤー数連動）
| 人数 | 飲食店/メディア/バス | 金融機関 | 株上限 |
|------|----------------------|----------|--------|
| 1〜4人 | 各3枚 | 1枚 | プレイヤー数 |
| 5〜8人 | 各6枚 | 2枚 | プレイヤー数 |
| 9〜10人 | 各8枚 | 3枚 | プレイヤー数 |

### ATM・イベント
- **ATM利息**（年末に自動付与）: 残高3500円〜→+600円 / 2600円〜→+300円 / 1500円〜→+200円 / 200円〜→+100円
- **イベントカード** 10種: インフレ・デフレ・株高騰・株暴落・倒産・利息UP・サイコロ偶奇固定・投資家イベントなど
- **破産システム**: 現金がマイナスになると破産・以降のターンに参加不可

### AI プレイヤー
- 会社・株の在庫や所持金を考慮した自動行動
- サイコロ先振り対応（`diceRolled` フラグに完全準拠）
- メディアボーナス自動徴収、エラー時の安全なターン終了

## 🎯 ゲームのポイント（教育効果）

1. **現金だけ持っていても増えない** → ATM・株・会社への分散投資の重要性
2. **リスクとリターンのトレードオフ** → 外国株は高リターン・高リスク
3. **複利の力** → ATM 残高が多いほど利息が増える
4. **タイミングと戦略** → イベントカードへの備え・売却判断
