# 💰 もしもし投資ランド！

子ども（7歳〜）が楽しく投資とお金の仕組みを学べるインタラクティブな投資シミュレーションゲームです。

## プロジェクト概要
- **名前**: もしもし投資ランド！
- **対象**: 7〜12歳の子どもとその保護者
- **目的**: ゲームを通じて投資・お金の基本概念を楽しく学ぶ

## 主な機能

### ゲームプレイ
- 🎮 **ターン制シミュレーション**（最大15ターン）
- 📈 **株式投資**: 6種類の株を売買、株価はランダムに変動
- 🏠 **不動産投資**: 5種類の物件を購入、毎ターン家賃収入
- 🎴 **イベントカード**: 14種類のランダムイベント（幸運・不運・チャンス）
- 🤖 **AI対戦**: 最大3人のAIプレイヤーと対戦
- 🏆 **ランキング・勝敗判定**: 最終資産額で順位決定

### 教育機能
- 📖 **チュートリアル**: 5ステップのわかりやすい説明
- 📚 **投資用語辞典**: 12の基本用語を解説
- 💡 **ゲーム終了後の学習ポイント**: 今日学んだことをまとめて表示

### UI/UX
- ✨ **カラフルなデザイン**: 子ども向けの明るく楽しいデザイン
- 🪙 **コインエフェクト**: 購入・売却時のアニメーション演出
- 📱 **レスポンシブ対応**: PC・タブレット・スマホ対応
- 🎯 **アクション確認モーダル**: 誤操作防止の確認画面

## URL
- **ローカル開発**: http://localhost:3000

## データ構造
- **プレイヤー**: 名前・現金・株保有数・不動産リスト・総資産
- **株式**: 6銘柄（テクノロジー・フード・エネルギー・ゲーム・ヘルス・エコ）
- **不動産**: 5物件（小さな家・マンション・お店・ホテル・農場）
- **イベントカード**: 14種類（幸運7・不運5・チャンス2）

## ゲームの遊び方
1. 名前を入力してゲームをスタート
2. 手持ちの¥50,000で株や不動産に投資
3. ターン終了でイベントカードが発生
4. 設定ターン数終了後、総資産が一番多い人の勝ち！

## 技術スタック
- **フレームワーク**: Hono (TypeScript)
- **デプロイ**: Cloudflare Pages
- **フロントエンド**: TailwindCSS (CDN) + Vanilla JS
- **フォント**: Nunito (Google Fonts)
- **アイコン**: FontAwesome

## 開発コマンド
```bash
npm run build    # ビルド
pm2 start ecosystem.config.cjs  # サービス起動
pm2 logs webapp --nostream       # ログ確認
```
